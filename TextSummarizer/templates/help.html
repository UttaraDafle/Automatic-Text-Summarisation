Welcome !

This is a tool to quickly summarize your text through url or files by condensing it into a shorter version while preserving the salient features.

This tool also provides analysis features:
1) Estimated Reading Time :It is based on the average reading speed of an adult (roughly 265 words per minute).
2)Length of text: You can easily view the length of input textand the resultant summary.

Compare Summarizers: Makes use of several methods to summarize text such as:
1. SpaCy Summarizer: spaCy is a open-source advanced natural language processing library.It helps in applications that process and understand large volumes of text. It can be used to build information extraction or natural language understanding systems, or to pre-process text for deep learning.
2. Text Rank summarizer:This algorithm is based from Google's Page Rank algorithm.is a graph-based ranking model for text processing which can be used in order to find the most relevant sentences in text and to find keywords.
3. NLTK Summarizer: The Natural Language Toolkit (NLTK) is a platform used for building Python programs that work with human language data for applying in statistical natural language processing. It contains text processing libraries for tokenization, parsing, classification, stemming, tagging and semantic reasoning.
4. LexRank Summarizer: LexRank is an unsupervised graph-based approach for automatic text summarization. It is used for computing sentence importance based on the concept of eigenvector centrality in a graph representation of sentences.

About Summary App

Using Spacy,NLTk, Text Rank and Lex Rank methods for Text Summarization.Also provides a Translator and Comparison of Methods functionality
